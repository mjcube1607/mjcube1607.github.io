CTFs Walkthrough

Below are the walkthroughs of various Capture The Flag (CTF) challenges that I have completed:

  • Agent Sudo: Web App, FTP, Binwalk, John, and Steghide. Read Walkthrough.
  • Basic Pentesting: enum4Linux, Hydra, LinEnum, John SSH, and id_rsa. Read Walkthrough.
  • Kenobi: SMB Enumeration, NFS, Proftpd 1.3.5 vulnerability, SITE CPFR CPTO, and id_rsa. Read Walkthrough.
  • Steel Mountain: Rejetto HTTP File Server vulnerability, Metasploit, PowerUp.ps1, Unquoted Service Paths, and `sc start stop`. Read Walkthrough.
  • Cyborg: Web Directory Enumeration, Borg backup recovery, Hashcat, SSH, `sudo -l`, and escalating privileges through altering backup files with sudo permissions. Read Walkthrough.
